# Visão Geral do Ascendya Manager

O Ascendya Manager é o sistema de criação, edição e organização dos conteúdos que alimentam o app mobile Ascendya. Ele permite a edição visual de capítulos, tópicos, tarefas e feedbacks, facilitando a organização de conteúdos gamificados e educativos para o aplicativo.

## Relação com o App Ascendya
O conteúdo produzido no Ascendya Manager é exportado em arquivos JSON e consumido diretamente pelo aplicativo Ascendya, garantindo padronização e facilidade de atualização dos conteúdos.

## Público-alvo
- Autores de conteúdo educacional
- Terapeutas e coaches
- Equipes de desenvolvimento do app Ascendya

## Principais Funcionalidades
- Edição visual de arquivos JSON estruturados
- Organização por áreas do conhecimento
- Suporte a imagens e feedbacks
- Interface intuitiva e responsiva

## Benefícios
- Padronização de conteúdos para o app
- Facilidade de uso para não programadores
- Exportação e importação de dados
