# Fluxo da Aplicação

1. **Início**: O usuário abre o Ascendya Manager.
2. **Carregar JSON**: Seleciona um arquivo de área do conhecimento para editar.
3. **Edição**: Altera capítulos, tópicos, tarefas e feedbacks visualmente.
4. **Salvar**: Exporta as alterações para um novo arquivo JSON.
5. **Imagens**: Pode associar imagens aos tópicos, salvas na pasta `images/`.
6. **Feedbacks**: Adiciona perguntas de feedback para cada capítulo.

## Diagrama Simplificado

```
[Início] -> [Carregar JSON] -> [Edição Visual] -> [Salvar JSON]
                                 |
                                 v
                            [Adicionar Imagens]
                                 |
                                 v
                            [Adicionar Feedbacks]
```
