# Guia de Desenvolvimento

## Tecnologias
- Electron (main.js, preload.js)
- React (App.js)

## Estrutura de Pastas
- `data/`: arquivos JSON de áreas
- `images/`: imagens associadas aos tópicos
- `components/`: componentes React (pode ser expandido)

## Scripts
- `npm install`: instala dependências
- `npm run dev`: executa o app em modo desenvolvimento

## Extensões sugeridas
- ESLint
- Prettier

## Sugestões de expansão
- Adicionar autenticação de usuários
- Suporte a múltiplos idiomas
- Exportação para outros formatos (PDF, HTML)
