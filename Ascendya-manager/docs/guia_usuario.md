# Guia do Usuário

## Abrindo um arquivo
- Clique em "Carregar JSON" e selecione um arquivo na pasta `data/`.

## Editando conteúdos
- Altere títulos, descrições, tópicos e tarefas diretamente na interface.
- Para adicionar imagens, selecione o arquivo desejado e associe ao tópico.
- Para adicionar perguntas de feedback, edite a seção de feedback do capítulo.

## Salvando alterações
- Clique em "Salvar JSON" para exportar o conteúdo editado.

## Recomendações
- Mantenha o padrão de estrutura dos arquivos para garantir compatibilidade.
- Utilize imagens otimizadas na pasta `images/`.
